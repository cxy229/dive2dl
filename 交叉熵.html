交叉熵

为什么要用交叉熵？

交叉熵只关心对正确类别的预测概率

比如，一个模型预测得到的概率分布：0.8，0.1，0.1，另一个模型得到的概率分布：0.8，0.2，0
最后的预测结果都是第0个类别，不管其他两个预测值为多少，类别预测均正确。

交叉熵更适合用来衡量两个概率分布差异

pytorch实现

y_hat: (batch_size, num_outputs)  # num_outputs是类别数
y: (batch_size, )
def cross_entropy(y_hat, y):
    return - torch.log(y_hat.gather(1, y.view(-1, 1)))
